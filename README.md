# Data-Structure-Code with C++

## 介绍
​		本项目根据严蔚敏等老师撰写的《数据结构》C语言版第2版的内容，使用C++语言分章节地实现了书中相关的数据结构和算法。

​		使用本书作为参考书考研并使用C++的同学可以在学习的途中参考本项目实现的代码。所有通用的抽象数据结构（ADT）的具体实现都使用了C++的模板类，用户可以自定义类型进行测试。
